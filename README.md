# LobsterThis is a repo for managing machine readable artifacts for the [Lobster APIs](https://lobster.media). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Lobster APIs](https://lobster.media).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://lobster.media)- [Developer](https://lobster.media)- [Twitter](https://twitter.com/Lobster_it)The [APIs.json](https://github.com/api-evangelist/lobster/blob/master/apis.json) index provides links to all the relevant links for the Lobster APIs, including the OpenAPI and Postman if they are present.
